How to save keys permanently (so you never enter them again)
Create files in ~/.keys/ — one key per file, no newlines:
mkdir -p ~/.keys && chmod 700 ~/.keys
echo -n "sk-your-openai-key" > ~/.keys/openai
echo -n "sk-your-deepseek-key" > ~/.keys/deepseek
echo -n "sk-ant-your-anthropic-key" > ~/.keys/anthropic
chmod 600 ~/.keys/*
Restart the server after adding keys. They load automatically on startup.
